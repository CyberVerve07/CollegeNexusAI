# **App Name**: CollegeNexus AI

## Core Features:

- Role-Based Login: Secure login system with distinct dashboards and access rights for Super Admin, Teacher, and Student roles.
- Attendance Management: Teachers can mark attendance (Present, Absent, Late) for students in selected courses, semesters, and sections. Students can view their attendance records.
- Timetable Display: Students can view their personalized timetable.
- Study Materials Repository: Teachers can upload study materials (PDF, images) for their subjects. Students can view and download these materials.
- Previous Year Question Papers: Admin and teachers can upload previous year question papers, and students can filter and download them based on course, semester, subject, exam type, and year.
- AI Chatbot Assistant: An AI chatbot powered by GPT-4o that provides students with study help, topic explanations, and syllabus-based answers; it also serves as a tool for teachers' admin/process queries.
- Admin Controls: Super Admin can manage courses, batches, semesters, sections, subjects, teacher accounts, student data (single or bulk upload), and global notices.

## Style Guidelines:

- Primary color: Deep blue (#3F51B5) to convey trust, intelligence, and stability.
- Background color: Light gray (#F5F5F5) to provide a clean, uncluttered interface.
- Accent color: Teal (#009688) to highlight interactive elements and call-to-action buttons.
- Headline font: 'Space Grotesk' sans-serif for headlines. Body font: 'Inter' sans-serif for main text.
- Code font: 'Source Code Pro' for displaying code snippets (if applicable).
- Use a set of modern, consistent icons for navigation and actions. Ensure icons are intuitive and easily recognizable.
- Subtle animations and transitions to provide feedback on user interactions and enhance the overall user experience. For example, smooth transitions between screens and loading animations.
